# VelaFlow — Project Start

**Project inception date:** 2026-04-17

VelaFlow is a self-hosted AI productivity system integrating Todoist, Notion,
Google Calendar, Gmail, and Google NotebookLM into a single automated pipeline.

Designed, architected, and implemented as a full-stack Python project with
Zero-Trust security, multi-model LLM orchestration, and infrastructure-as-code
deployment on Proxmox LXC.
